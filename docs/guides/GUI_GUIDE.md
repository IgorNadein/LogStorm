# GUI Guide

GUI-разработка приостановлена. Код сохранен в `gui/` и запускается через `run_gui.py`, но сейчас не является основным пользовательским путем и не входит в обязательный pytest/smoke контур.

Основной путь работы:

```bash
python -m pytest
python main.py
```

Перед возвратом к GUI нужно отдельно решить:

- оставляем ли PySide6/qfluentwidgets;
- переносим ли GUI на актуальные core services;
- какие GUI-сценарии покрываются тестами;
- нужен ли GUI вообще, если CLI/collector покрывают основной workflow.
