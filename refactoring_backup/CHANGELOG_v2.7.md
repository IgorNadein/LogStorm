# 🎉 LogStorm v2.6 → v2.7 - Итоговая сводка изменений

## 📋 Что было добавлено

### 1. 📚 Множественный выбор файлов

#### ✅ DataLoader
- Метод `load_logs()` теперь принимает `str` или `list`
- Новый метод `_load_multiple_files()` для объединения
- Новый метод `_load_single_file()` для загрузки одного файла
- Автоматическая сортировка по timestamp
- Обработка ошибок для каждого файла отдельно

#### ✅ GUI (gui_app.py)
- `logs_path` → `logs_paths` (список вместо строки)
- Новая кнопка "➕ Добавить файл"
- Новая кнопка "🗑️ Очистить всё"
- Скроллируемый Listbox для отображения файлов
- Методы: `_add_log_file()`, `_clear_log_files()`, `_update_logs_list()`
- Обновлённая валидация (проверка списка файлов)

### 2. 🔓 Опциональные профили

#### ✅ DataLoader
- `load_preferences()` теперь возвращает `{}` если файл не найден
- Параметр `path` теперь опциональный (`path=None`)
- `filter_known_users()` пропускает фильтрацию если `prefs={}` 
- Информативные сообщения о работе без профилей

#### ✅ AttendanceService
- `analyze_all()` берёт всех пользователей из логов если `prefs` пустой
- Работает с дефолтными графиками (08:00-17:00, Пн-Пт)

#### ✅ GUI (gui_app.py)
- Поле профилей помечено "(необязательно)"
- Диалог подтверждения если файл не найден
- Возможность продолжить без профилей
- Подсказка "💡 Если не указан - используются дефолтные настройки"

#### ✅ main.py
- Проверка существования файла перед загрузкой
- Fallback к пустому словарю

## 🎯 Новые возможности

### Множественные файлы:

**До (v2.6):**
```python
df = DataLoader.load_logs("logs/attendance.csv")
```

**После (v2.7):**
```python
# Один файл (обратная совместимость)
df = DataLoader.load_logs("logs/attendance.csv")

# Несколько файлов (НОВОЕ!)
files = [
    "logs/october.csv",
    "logs/november.csv",
    "LogsCam/events.ndjson"
]
df = DataLoader.load_logs(files)
```

### Опциональные профили:

**До (v2.6):**
```python
# ОБЯЗАТЕЛЬНО нужен person_prefs.json
prefs = DataLoader.load_preferences("path/person_prefs.json")
# FileNotFoundError если файл отсутствует
```

**После (v2.7):**
```python
# Можно без файла
prefs = DataLoader.load_preferences(None)  # Вернёт {}

# Можно с несуществующим путём
prefs = DataLoader.load_preferences("nonexistent.json")  # Вернёт {}

# Анализируются ВСЕ пользователи с дефолтами
```

## 🔧 API изменения

### DataLoader.load_logs()
```python
# Было (v2.6):
def load_logs(path: str, file_type: str = 'auto') -> pd.DataFrame

# Стало (v2.7):
def load_logs(path, file_type: str = 'auto') -> pd.DataFrame
# path: str | list[str]
```

### DataLoader.load_preferences()
```python
# Было (v2.6):
def load_preferences(path: str) -> dict
# Исключение если файл не найден

# Стало (v2.7):
def load_preferences(path: str = None) -> dict
# Возвращает {} если path=None или файл не найден
```
