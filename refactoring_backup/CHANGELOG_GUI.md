## 🚀 Как использовать

### GUI режим (рекомендуется)
```bash
python run_gui.py
```

**Шаги:**
1. Выберите файл логов (CSV или NDJSON)
2. Выберите файл профилей
3. Настройте опции
4. Нажмите "Запустить анализ"
5. Наблюдайте за процессом
6. Откройте готовый отчёт

### CLI режим (как раньше)
```bash
python main.py
```

## 🎯 Поддерживаемые форматы

### CSV (camera logs)
```csv
timestamp,camera,name,distance,identity
2025-10-01T15:59:57,main_entrance,igor_nadein,0.34107,"photo.jpg"
```

### NDJSON (Hikvision СКУД)
```json
{"major": 5, "minor": 75, "time": "2025-12-01T06:05:04+03:00", "name": "Иван"}
{"major": 5, "minor": 104, "time": "2025-12-01T17:05:13+03:00", "name": "Иван"}
```

**Автоопределение:**
- По расширению: `.csv`, `.ndjson`, `.jsonl`
- По содержимому: первая строка начинается с `{`

## 🔧 Архитектура

```
GUI (gui_app.py)
    ↓
DataLoader (автоопределение формата)
    ↓
├── CSV → pandas DataFrame
└── NDJSON → LogsCamLoader → EventMapper → pandas DataFrame
    ↓
AttendanceService (анализ)
    ↓
ExcelReporter + SummaryReporter + AIService
    ↓
Отчёты (Excel + консоль + AI)
```

## 🐛 Известные ограничения

1. GUI требует основной поток для tkinter
2. NDJSON парсер загружает весь файл в память
3. Большие файлы (>100k событий) могут быть медленными
4. AI анализ требует интернет и GigaChat API
