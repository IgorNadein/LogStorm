# GUI Quick Start

## 🚀 Быстрый старт

### Запуск GUI
```bash
python run_gui.py
```

### Использование

1. **Вкладка "⚙️ Настройки"**
   - **Файлы логов**: Добавьте один или несколько файлов (CSV или NDJSON)
   - **Профили сотрудников**: Выберите person_prefs.json (необязательно для NDJSON)
   - **Маппинг сотрудников**: Выберите person_mapping.json (для NDJSON с маппингом)
   - **Файл отчёта**: Укажите путь для сохранения Excel
   - **Опции**: Включите AI анализ при необходимости
   - Нажмите "▶️ Запустить анализ"

2. **Вкладка "📊 Анализ"**
   - Наблюдайте за процессом в реальном времени
   - Видите прогресс по этапам
   - После завершения откройте отчёт

3. **Вкладка "ℹ️ О программе"**
   - Информация о возможностях
   - Поддерживаемые форматы

## 🎯 Типичные сценарии

### Сценарий 1: Анализ CSV логов
1. Запустите GUI
2. Файл логов: `logs/attendance.csv`
3. Тип файла: CSV или Авто
4. Запустите анализ

### Сценарий 2: Анализ СКУД NDJSON с маппингом
1. Запустите GUI
2. Добавьте файлы: `LogsCam/vhod.ndjson`, `LogsCam/vihod.ndjson`
3. Тип файла: NDJSON или Авто
4. Файл маппинга: `person_mapping.json`
5. Запустите анализ
   - Имена будут нормализованы (Серёга → Сергей)
   - Алиасы объединены
   - Индивидуальные расписания применены

### Сценарий 2б: NDJSON без маппинга
1. Запустите GUI
2. Файл логов: `LogsCam/events_with_pic.ndjson`
3. Тип файла: NDJSON или Авто
4. Оставьте маппинг пустым или удалите путь
5. Запустите анализ (будут использованы оригинальные имена)

### Сценарий 3: Переключение источника
- Просто выберите другой файл через "Обзор..."
- Формат определится автоматически

## ⚙️ Настройки по умолчанию

Все пути можно изменить в `gui_config.py`:
```python
DEFAULT_CSV_PATH = "logs/attendance.csv"
DEFAULT_NDJSON_PATH = "LogsCam/events_with_pic.ndjson"
DEFAULT_PREFS_PATH = "path/person_prefs.json"
DEFAULT_OUTPUT_PATH = "reports/attendance_report.xlsx"
```

## 🐛 Решение проблем

### GUI не запускается
- Убедитесь что установлен Python 3.8+
- Установите зависимости: `pip install -r requirements.txt`
- tkinter уже включен в Python (на Windows)

### Файлы не найдены
- Используйте кнопку "Обзор..." для выбора файлов
- Проверьте пути в статус-баре

### AI анализ не работает
- Убедитесь что настроен `.env` файл
- Проверьте GigaChat API ключ
- Или отключите AI анализ в настройках

## 📝 Отличия от CLI

| Параметр | CLI | GUI |
|----------|-----|-----|
| Выбор файлов | Жёстко в коде | Через диалоги |
| Прогресс | Консоль | Прогресс-бар + лог |
| Отчёт | Автоматически | Кнопка "Открыть" |
| Настройки | Код/конфиг | Чекбоксы/радио |
