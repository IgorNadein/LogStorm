### 3. 🔄 Параллельные запросы (требует модификации скрипта)


### Шаг 3: Разделить на периоды

Для полного года создайте скрипт:

```python
# split_export.py
import subprocess
from datetime import datetime, timedelta

start = datetime(2025, 1, 1)
end = datetime(2025, 12, 19)
chunk_days = 30

current = start
files = []

while current < end:
    chunk_end = min(current + timedelta(days=chunk_days), end)
    
    filename = f"events_{current.strftime('%Y%m%d')}_{chunk_end.strftime('%Y%m%d')}.ndjson"
    
    cmd = [
        "python", "export_acs_events.py",
        "--host", "192.168.1.101",
        "--user", "admin",
        "--password", "xZ260616737",
        "--start", current.strftime("%Y-%m-%dT00:00:00"),
        "--end", chunk_end.strftime("%Y-%m-%dT23:59:59"),
        "--pic",
        "--out", filename
    ]
    
    print(f"Экспорт {current.date()} - {chunk_end.date()}...")
    subprocess.run(cmd, check=True)
    files.append(filename)
    
    current = chunk_end

# Объединить файлы
with open("vhod_full.ndjson", "w", encoding="utf-8") as out:
    for f in files:
        with open(f, "r", encoding="utf-8") as inp:
            out.write(inp.read())

print(f"Готово! Объединено {len(files)} файлов в vhod_full.ndjson")
```
