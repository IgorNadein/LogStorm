# LogStorm Fluent GUI - Руководство пользователя

## Обзор

LogStorm теперь имеет современный интерфейс в стиле Windows 11 Fluent Design с использованием PySide6-Fluent-Widgets.

## Основные возможности

### 1. Настройки
**Вкладка:** Настройки (иконка шестеренки)

- **Добавить файл** - добавление файлов логов (.csv, .ndjson)
- **Очистить** - очистка списка файлов
- **Использовать AI анализ** - включение/выключение GigaChat для анализа причин отсутствия
- **Применить настройки** - сохранение выбранных файлов и опций

### 2. Сотрудники
**Вкладка:** Сотрудники (иконка группы)

Управление персональными настройками сотрудников:
- **Обновить список** - перезагрузить person_prefs.json
- **Добавить** - добавить нового сотрудника
- **Редактировать** - изменить настройки выбранного сотрудника
- **Удалить** - удалить сотрудника из базы

#### Поля сотрудника:
- **ID** - уникальный идентификатор (ivan_ivanov)
- **Имя** - отображаемое имя (Employee Sample)
- **Начало работы** - время начала рабочего дня (09:00)
- **Рабочие дни** - список рабочих дней через запятую (Monday, Tuesday, ...)

### 3. Анализ
**Вкладка:** Анализ (иконка графика)

Выполнение анализа логов посещаемости:
- **Запустить анализ** - начать обработку файлов
- **Индикатор прогресса** - показывает текущий этап (загрузка, анализ, AI)
- **Результаты** - статистика по обработанным записям и статусам

Процесс анализа:
1. Загрузка файлов
2. Анализ посещаемости
3. AI анализ отсутствий (если включено)
4. Отображение статистики

### 4. Экспорт
**Вкладка:** Экспорт (иконка поделиться)

Экспорт результатов анализа:
- **Экспорт в Excel (.xlsx)** - основной формат с форматированием
- **Экспорт в CSV** - простой текстовый формат
- **Экспортировать** - выбор места сохранения и создание отчёта

### 5. О программе
**Вкладка:** О программе (иконка информации, внизу)

Информация о версии и возможностях LogStorm.

## Рабочий процесс

### Типичный сценарий использования:

1. **Настройка файлов**
   - Перейдите в "Настройки"
   - Нажмите "Добавить файл" и выберите файлы логов
   - Выберите опцию "Использовать AI анализ" при необходимости
   - Нажмите "Применить настройки"

2. **Проверка сотрудников**
   - Перейдите в "Сотрудники"
   - Убедитесь что все нужные сотрудники загружены
   - При необходимости добавьте или отредактируйте записи

3. **Запуск анализа**
   - Перейдите в "Анализ"
   - Нажмите "Запустить анализ"
   - Дождитесь завершения (прогресс отображается в реальном времени)
   - Просмотрите статистику в списке результатов

4. **Экспорт отчёта**
   - Перейдите в "Экспорт"
   - Выберите формат экспорта
   - Нажмите "Экспортировать"
   - Выберите место сохранения файла

## Технические детали

### Архитектура
- **UI Framework:** PySide6 (Qt6)
- **Design System:** Windows 11 Fluent Design (PySide6-Fluent-Widgets)
- **Threading:** QThread для фонового анализа
- **State Management:** AppState класс для централизованного состояния

### Компоненты
- **FluentWindow** - главное окно с навигацией
- **CardWidget** - карточки для группировки элементов
- **IndeterminateProgressBar** - анимированный прогресс
- **InfoBar** - всплывающие уведомления
- **MessageBox** - диалоги подтверждения

### Файлы
- `gui_app_fluent.py` - основной код GUI
- `run_gui.py` - точка входа
- `path/person_prefs.json` - настройки сотрудников

## Преимущества нового интерфейса

✅ **Нативный вид** - полное соответствие Windows 11 Fluent Design  
✅ **Отзывчивость** - фоновая обработка не блокирует интерфейс  
✅ **Удобство** - интуитивная навигация и понятные элементы  
✅ **Информативность** - подробные статусы и уведомления  
✅ **Современность** - актуальные компоненты и стили  

## Требования
- Windows 10/11
- Python 3.10+
- PySide6 >= 6.6.0
- PySide6-Fluent-Widgets >= 1.10.0

## Известные ограничения
- AI анализ требует настроенный GigaChat (см. GIGACHAT_SETUP.md)
- Большие файлы могут требовать времени на обработку
- Тёмная/светлая тема определяется автоматически по системе
